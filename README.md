# Personal Portfolio Web App — Dog Breed Personality Quiz | Python, Streamlit

- Built a multi-page personal portfolio web app using Streamlit, showcasing education, experience, projects, and programming skills.
- Designed a customizable portfolio page driven by a centralized data file (`info.py`), separating content from presentation for easy updates.
- Developed an interactive dog breed personality quiz that recommends a dog breed based on the user's answers.
- Integrated external profile links (LinkedIn, GitHub, email) with icon-based navigation for a polished user experience.

**Achievements:**

- Developed a multi-page Streamlit application with sidebar navigation
- Implemented a data-driven portfolio layout using Python dictionaries for education, experience, projects, skills, and activities
- Built an interactive personality quiz with conditional logic for dynamic breed recommendations
- Structured code with separation of concerns between UI pages and content data
